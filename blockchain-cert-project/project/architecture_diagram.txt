|   User Interface  |       |   Script.js       |       |   Blockchain.js   |

| - Verify Input    |       | - Verify Hash     |       |                   |

                                                        |    isChainValid)  |
                                                        +-------------------+
                                                                 |
                                                                 |
                                                                 v
                                                        +-------------------+
                                                        |   Crypto Library  |
                                                        |   (crypto-js)     |
                                                        |                   |
                                                        | - SHA-256 Hashing |
                                                        +-------------------+

Flow:
1. User interacts with UI (upload, generate, add, verify).
2. Script.js processes events, calls Blockchain.js for chain operations.
3. Blockchain.js uses crypto-js for hashing and manages blocks.
4. Results displayed back in UI.

Key Components:
- UI: Pure HTML/CSS for layout and styling.
- Logic: JS handles interactions and blockchain logic.
- Hashing: External library for SHA-256 (no server needed).
